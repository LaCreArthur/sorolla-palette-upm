# Sorolla Palette

A complete mobile publisher SDK for Unity games.

## Features

- **Prototype Mode**: GameAnalytics + Facebook SDK + optional MAX (for CPI testing)
- **Full Mode**: GameAnalytics + MAX + Adjust (for production)

## Installation

### Automatic Installation (Recommended)

1. Open Unity Package Manager (`Window > Package Manager`)
2. Click `+` → `Add package from git URL`
3. Enter: `https://github.com/yourusername/sorolla-palette.git`
4. Click **Add**

**That's it!** The package will automatically:
- ✅ Configure scoped registries (OpenUPM)
- ✅ Install GameAnalytics SDK
- ✅ Install External Dependency Manager
- ✅ Set up the package structure

Check the Console for setup progress. Unity Package Manager will resolve dependencies automatically (1-2 minutes).

## Quick Start

1. Open `Tools > Sorolla Palette > Configure`
2. Select your mode (Prototype or Full)
3. Follow the configuration wizard
4. Import required manual SDKs (Facebook/Adjust)
5. Configure your API keys
